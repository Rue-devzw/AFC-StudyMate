# AFC StudyMate Blueprint

## Overview

AFC StudyMate is a mobile application designed to assist members of the Apostolic Faith Church (AFC) in their daily spiritual nourishment. It provides easy access to the Bible, Sunday school lessons, and other study materials. The app aims to be a comprehensive and user-friendly tool for personal and group study.

## Implemented Features & Design

### Core Features

*   **Bible Reader:**
    *   Browse and read different versions of the Bible (KJV, ASV, etc.).
    *   Navigate by book and chapter.
    *   "Continue Reading" feature to quickly return to the last read chapter.
*   **Sunday School Lessons:**
    *   View a list of all available Sunday school lessons.
    *   Filter lessons by class (e.g., Adult, Youth, Primary).
    *   Detailed lesson view including objectives, scriptures, content, and teacher notes.
    *   Mark lessons as complete to track progress.
*   **Verse of the Day:**
    *   Displays an inspiring Bible verse on the home screen each day.
    *   Share the verse with others.
*   **Settings:**
    *   Toggle between light and dark mode.
    *   Select the preferred Bible version.
    *   Reset reading progress and lesson completion status.

### Design & UI

*   **Theme:**
    *   Modern Material 3 design.
    *   Primary color: Deep Purple.
    *   Light and dark color schemes.
    *   Custom typography using `google_fonts` (Oswald, Roboto, Open Sans).
*   **Home Screen:**
    *   Prominent display of the AFC StudyMate logo.
    *   "Verse of the Day" card with a share button.
    *   Buttons for "Read Bible" and "Continue Reading".
*   **Bible Screen:**
    *   A clean list of all books of the Bible.
    *   Staggered animation for a visually appealing entrance.
*   **Book Screen:**
    *   A grid view of chapters for the selected book.
*   **Chapter Screen:**
    *   A clear presentation of the verses in the selected chapter.
    *   Scroll restoration to remember the user's position.
*   **Lessons Screen:**
    *   A tabbed view to filter lessons by class.
    *   Each lesson is displayed in a card with its title and a completion checkmark.
*   **Lesson Detail Screen:**
    *   A detailed and well-structured view of the lesson content.
    *   A floating action button to mark the lesson as complete.

## Current Plan

### Next Steps

1.  **Run the application:** Launch the app on a device or emulator to verify that all recent changes are working correctly and the app is stable.
2.  **User Testing and Feedback:** Gather feedback from users on the current features and identify any areas for improvement or new feature requests.
3.  **Future Enhancements (Potential):**
    *   Search functionality for the Bible and lessons.
    *   Bookmarking favorite verses and lessons.
    *   Adding personal notes to lessons and verses.
    *   Integration with other AFC resources (e.g., hymns, audio sermons).
